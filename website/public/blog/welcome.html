Welcome to My New Site

2024-03-03

Welcome to my new personal website! As the Head of Cybersecurity Engineering at Tesco, I needed a space that prioritized a few core things:

  1. Security: A static site with no backend databases to exploit and strong CSP headers.
  2. Speed & Minimalism: No bloated JavaScript frameworks, just pure, semantic HTML and minimal CSS.
  3. Ease of Use: A system where I can write draft posts in Markdown directly inside my IDE.

Why Markdown?

Markdown allows me to focus purely on the text without wrestling with a WYSIWYG editor. Writing technical content often involves code snippets, which Markdown handles beautifully:

def secure_function():
    print("Security first!")
    return True
